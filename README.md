
# Data Warehousing Project
# Overview
This project implements an end-to-end Data Warehousing solution using SQL Server, covering ETL processes, data modeling, and analytics to transform raw data into meaningful insights for decision-making.

# Tech Stack
Database: Microsoft SQL Server
ETL: SQL Server Integration Services (SSIS) 

# Data Modeling
Fact Tables: Store measurable business metrics
Dimension Tables: Store descriptive attributes
Schema Design: Optimized for analytical queries
Keys: Surrogate keys used for dimensions

# ETL PROCESS
Extract data from source systems
Clean and validate data in staging tables
Transform data (data type conversion, deduplication, aggregation)
Load data into dimension and fact tables
Incremental and full load strategies implemented
